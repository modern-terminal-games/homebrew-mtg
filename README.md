# homebrew-mtg

Homebrew tap for [**mtg**](https://github.com/modern-terminal-games/mtg).

## Install

```bash
brew tap modern-terminal-games/mtg
brew trust modern-terminal-games/mtg
brew install mtg
```

```bash
mtg list
mtg breaker
mtg paddle-battle
```

## Upgrade

```bash
brew update && brew upgrade mtg
```
