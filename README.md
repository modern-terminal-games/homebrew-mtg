# homebrew-mtg

Homebrew tap for [**mtg**](https://github.com/modern-terminal-games/mtg) — modern terminal games.

## Install

```bash
brew tap modern-terminal-games/mtg
brew trust modern-terminal-games/mtg
brew install mtg
```

```bash
mtg
mtg list
mtg breaker
```

## Upgrade

```bash
brew update && brew upgrade mtg
```
